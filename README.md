# visa-approval-project
visa approval project

- Data link: https://www.kaggle.com/datasets/moro23/easyvisa-dataset

## how to run?


   ```bash
   conda create -n visa python=3.8 -y
   ```

   ```bash
   conda activate visa
   ```

   ```bash
   pip install -r requirements.txt
   ```
